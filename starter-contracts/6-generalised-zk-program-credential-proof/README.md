# Generalised Credential Issuance & Attestation

This is a basic Typescript example of how to construct verifiable credentials about a subject.

## How to build

```sh
npm run build
```

## How to run tests

```sh
npm run test
npm run testw # watch mode
```

## How to run coverage

```sh
npm run coverage
```
